## Prerequisites
You need to have [Node.js and npm](https://nodejs.org/en/)
- Support Node v4 - latest
- Support npm v3 - latest

## Installation

Install:
```bash
npm install
```

## NPM commands
Start development server:
```bash
npm start
```
Build app for development:
```bash
npm run build
```
Start production server:
```bash
npm run start:prod
```
Build app for production:
```bash
npm run build:prod
```
